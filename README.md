# Chebyshev Bandpass IIR Filter

An **Arduino Uno R4 WiFi–based Chebyshev Type I IIR bandpass filter** implementation.

---

## Specifications

- **Passband:** 10 kHz – 20 kHz  
- **Cutoff Frequency 1:** 10 kHz  
- **Cutoff Frequency 2:** 20 kHz  
- **Passband Ripple:** ~4.5 dB  

- **Stopband Frequency 1:** 8.5 kHz  
- **Stopband Frequency 2:** 21.5 kHz  
- **Stopband Attenuation:** ~40 dB  

---

## Platform

- **Microcontroller:** Arduino Uno R4 WiFi  
- **Filter Type:** Chebyshev Type I  
- **Structure:** IIR Bandpass
